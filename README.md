# 🔐 Cyber Risk Assessment Dashboard

## 📌 Project Overview
The Cyber Risk Assessment Dashboard is a web-based application developed using Python and Streamlit.  
It helps in identifying vulnerabilities, calculating risk scores, and visualizing cybersecurity threats.

## 🎯 Objectives
- Detect open ports
- Calculate risk scores
- Visualize threats
- Send email alerts

## ⚙️ Technologies Used
- Python
- Streamlit: https://streamlit.io  
- Plotly: https://plotly.com  
- Nmap: https://nmap.org  
- Pandas: https://pandas.pydata.org

## ▶️ How to Run

1. Install:
pip install streamlit pandas plotly

2. Run:
python main.py

3. Start dashboard:
streamlit run dashboard.py
